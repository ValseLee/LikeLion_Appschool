### 프로토콜
---

- 놀랍게도 objc에는 프로토콜이 있다.
- 특정 상황에 사용될 것으로 예상되는 메소드를 선언하는 프로토콜 정의가 가능하다.
    + 세상에 선택 구현부가 있다...
    + 채택하면 인터페이스에서 다시 선언할 필요는 없고 구현만 해주면 된다.
    + `delegate`는 이때부터 시작되었다..
```objc
@protocol ProtocolNameDelegate
@required 
// required methods
@optional
// optional methods
@end

@interface MyClass: NSObject <ProtocolNameDelegate>
// code
@end
```
